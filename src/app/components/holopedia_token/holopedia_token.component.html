Holopedia Token ~ Introduction

Holopedia Token has been created to serve as a payment method for holographic content that is shared among Holopedia platform users. Holopedia will incentivize users to upload holographic data of different kind by rewarding them Holopedia Tokens.

Holopedia Token is an ERC20 token and thus benefits from simple adaptivity, you just need to add the token to your favourite Ethereum wallet (search by token name or by contract key from Etherscan).

Holopedia Token is currently developeing holograms oriented DApps (HoloApps) that will facilitate gamification of the activities around hologram content browsing, sharing and use in specialized display devices.
