Holopedia Tokens Hybrid ~ Technical procedures

Holopedia Ecosystem

Holopedia ecosystem is built with tokens & coins developed on Distributed Ledger Technology (DLT). Holopedia Tokens are smart contracts within Ethereum network (ERC20) and Holopedia Coins are so called alt coins.

Both tokens and coins are deployed within DLT network and they are synchronized across the entire network. The synchronisation is handled by the Client Synchronization Protocol which is a secure mechanism the role of which is to update the server gets whenever any new transaction is made within the network. All new transactions within the network are automatically synchronized with clients that are connected to the network. The clients is ready to indroduce new transactions when first it is synchronized with the network.

Tokens & Coins distribution & exchanges structure

Holopedia Tokens Hybrid is developed as ERC20 smart contract and thus benefits from simple adoption. You can use your favorite Ethereum wallet and add the token either by the token name, or using a contract key that you can find on Etherscan.

Internal payment

Holopedia Tokens Hybrid has been created to serve as a payment method for holographic & 3D content that is shared among platform users. Holopedia project facilitates user based content and will rewarding them Holopedia Tokens & Coins for various activites on the network including signing up, refererring other users and uploading holographic & 3D contect.

Holopedia Tokens Hybrid Distributed Ledger Technology DLT

Distributed Ledger Technology is a decentrilized network that does not consist of one point of failure. Data is distributed and syncronized across the nodes where each node contain the cope of the entire ledger. This has several advantages over centralized network. It is not dependant on a central authority, both in terms of decision making and for the data safety. The security of the transactions within the network is validated by the whole network through its concensus algorithm.
